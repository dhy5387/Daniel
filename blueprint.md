# Lotto Number Generator

## Overview

This project is a simple web application that generates random lottery numbers.

## Style, Design, and Features

*   **UI:** The application will have a clean and modern user interface.
*   **Functionality:** Users can click a button to generate 6 unique random numbers between 1 and 45.
*   **Display:** The generated numbers will be displayed in a visually appealing way.

## Current Plan

1.  **Modify `index.html`:** Update the HTML structure to include a title, a container for the lottery numbers, and a button to generate them.
2.  **Modify `style.css`:** Add CSS to style the application, including the number display and the button.
3.  **Modify `main.js`:** Implement the JavaScript logic for generating and displaying the lottery numbers.
